# Supabase Setup

1. Criar um novo projeto Supabase.
2. Abrir o SQL Editor.
3. Executar `supabase-schema.sql`.
4. Ativar Authentication por email.
5. Copiar:
   - Project URL
   - anon public key

A app deve usar autenticação. Não deixar tabelas públicas com escrita anónima, porque o site fica publicado no GitHub/Netlify.
