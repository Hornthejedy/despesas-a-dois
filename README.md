# Despesas a Dois

Aplicação HTML para registar despesas partilhadas, calcular a divisão por duas pessoas e controlar liquidações.

## Funcionalidades

- Registo de data, local/descrição e valor pago.
- Filtro por mês.
- Ordenação por data, local, valor e estado.
- Marcação de despesas como **Oferta**.
- Ofertas ficam no histórico, mas não entram nas contas a dois.
- Registo de liquidações e data até à qual a dívida fica paga.
- Exportação CSV.
- PWA simples para usar no telemóvel.

## Publicação

O ficheiro principal é `index.html`.
